# Felhasználói útmutató

## Az alkalmazás célja
A RecipeBox célja receptek egyszerű nyilvántartása és kezelése webes felületen.

## Elérhető funkciók

### 1. Receptlista
A kezdőképernyőn megjelenik a receptek listája. A lista lapozható.

### 2. Új recept létrehozása
Az **Új recept** gombra kattintva űrlap nyílik meg, ahol megadható:
- recept neve
- rövid leírás
- kategória
- elkészítési idő
- hozzávalók
- elkészítési lépések

### 3. Recept részletei
A **Részletek** gombbal megtekinthető a recept teljes tartalma.

### 4. Recept szerkesztése
A **Szerkesztés** gombra kattintva a meglévő adatok módosíthatók.

### 5. Recept törlése
A **Törlés** gombra kattintva a recept eltávolítható.
